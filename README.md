# minecraftmusicmachine
Create note block sequences in minecraft with blocks under them. Enters commands into MCJE console by taking keyboard control.

`generate_piano.py` -> Generates minecraft commands to place a sequence of note blocks with given blocks under them, starting at a given position, in a given direction. For example input see `songs/`. For example outputs see `commands/`. For usage ask an LLM, it's 2026.

`enter_commands.py` -> Takes keyboard control and enters given list of commands into Minecraft JE console. For example inputs see `commands/`. For usage ask an LLM.
